This website is hand coded!

This website, my portfolio, doesn't use any fancy frameworks like bootstrap. No website builder or anything, the only outside element is google fonts! When building this out I wanted to challenge myself and make everything in just html, css, and js.

It was a challenge to get everything to work correctly. I have gotten used to working with tools such as bootstrap, so it was difficult for me to do things like keep the website responsive or make the images stay where they need to be.

Check out the source code on my github!
